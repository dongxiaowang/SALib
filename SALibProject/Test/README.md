# SALib
Libary for Selenium and Appium.

##### 仓库设置  

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://www.jitpack.io</url>
    </repository>
</repositories>
```

#### 导入项目
```xml
<dependency>
    <groupId>com.github.dongxiaowang</groupId>
    <artifactId>SALib</artifactId>
    <version>version</version>
</dependency>
```
